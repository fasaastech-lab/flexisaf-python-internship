# Reusable Python Script for Interactive User Profile Collection

## Project Context
Week 1 deliverable — FlexiSAF Python Programming Internship (Beginner Stage)

## Description
The program collects a user's profile, validates each input, saves it to a JSON file, and reads it back to the user.

## How to Run

Clone the repository:
```bash
git clone git@github.com:fasaastech-lab/flexisaf-python-internship.git
```

Navigate to the folder:
```bash
cd flexisaf-python-internship
```

Run the script:
```bash
python week1_deliverables.py
```

## Sample Output
Saved Profile:
Name: Abdulraheem
Age: 31
Email: fasaastech@gmail.com
Phone: 09012981955

## Requirements
- Python 3.6 or higher
- No external libraries required. Uses only Python built-in modules.
